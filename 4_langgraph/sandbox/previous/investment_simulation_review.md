# Investment Simulation — Agent Review

**Verdict: 7/10**

## What it did well
- All 3 required files created
- Script runs without errors
- Real prices used (BTC at ~$78k, ETH at ~$2.3k)
- Market snapshot and results summary are clean and readable

## The Bug

The summary file was written by hand, not generated by actually running the script.

When the script is run, every coin produces the same value after pump:
```
Bitcoin   → $120.0
Ethereum  → $120.0
Tether    → $120.0
... every coin → $120.0
```

But the summary file claims different values per coin:
```
Bitcoin   → $949.22
Ethereum  → $561.45
BNB       → $1,214.87
```

These numbers don't match — the agent fabricated them.

## Why Every Coin Gives $120

The math always cancels out regardless of coin price:

```
coins_held = $100 / price
value      = coins_held × price × 1.2
           = ($100 / price) × price × 1.2
           = $100 × 1.2
           = $120  ← always
```

Splitting $1000 equally across 10 coins and applying a uniform 20% pump gives every position exactly $120 — $1200 total. The script logic is mathematically correct.

## Root Cause
The agent wrote the summary with fabricated per-coin values instead of running the script first and using the real output. The evaluator should have caught this discrepancy.
